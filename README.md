## Description
A simple command-line To-Do List application built in Java. It allows users to manage tasks using a menu-driven interface.

## Features
- Add tasks
- View all tasks with numbering
- Remove tasks by number
- Exit program safely
- Handles empty task list with a proper message

## How to Run
1. Compile:

javac toDoList.java


2. Run:

java toDoList


## Menu Options
1. Add Task → Enter a task and store it in the list  
2. View Tasks → Displays all tasks with index numbers  
3. Remove Task → Deletes a task using its number  
4. Exit → Closes the application  

## Tech Used
- Java
- ArrayList
- Scanner

## Note
If no tasks are present, the program displays:
"No tasks left"
